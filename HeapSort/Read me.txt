1. Open the HeapSort.java file in any IDE of your preference.
2. Once it is opened in the IDE, run the program.
3. After the execution of the program, system prints the initial array before sorting and before heapify.
4. Then the system prints the modified array after heapify.
5. And then system prints the modified array after sorting.

NOTE:  

1. In the program , the variable 'n' holds the size of the array and the root of the heap is present at the index '0' .
2. The input array is { 12, 11, 13, 7, 15, 19, 21, 42, 53, 29, 28, 25, 36, 18, 17 }.
